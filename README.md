# QuietSettle

QuietSettle is a production-ready frontend and smart-contract reference for confidential milestone escrow on Zama FHEVM.

The app lets builders create encrypted escrow deals, fund Sepolia deposits, collect buyer and verifier approvals, release vendor payments, refund before approval, and sign EIP-712 decryption requests for authorized confidential amount access.

## Files

- `index.html` - production website and functional frontend
- `ConfidentialMilestoneEscrow.sol` - Zama FHEVM escrow contract
- `QuietSettle.md` - documentation and pitch notes
- `quietsettle-website-images/` - generated section references used by the website

## Deploy

This is a static site. Deploy the folder to Vercel, GitHub Pages, or any static host.
